# Background Animations

## Overview

The BIOCODE website now supports multiple psychologically-optimized background animations designed to build trust and stimulate action for weight loss supplement sales. These animations can be easily switched from the admin panel.

## Available Animations

### 1. Flowing Energy (Recommended)
**Психологически ефект:** Динамична енергия и метаболитен стимул

Плавно движещи се енергийни полета, които създават усещане за движение и трансформация. Подходяща за стимулиране на действие без да бъде прекалено агресивна.

**Характеристики:**
- Мека, непрекъсната анимация
- Подходяща за светли и тъмни теми
- Добре балансирана между динамика и спокойствие

### 2. Wellness Pulse
**Психологически ефект:** Доверие, здраве и виталност

Спокоен пулсиращ ефект, който напомня на сърдечен ритъм или дишане. Създава усещане за стабилност и надеждност.

**Характеристики:**
- Бавна, ритмична анимация
- Изгражда доверие
- Идеална за емоционално свързване

### 3. Metabolic Waves
**Психологически ефект:** Телесна трансформация и метаболизъм

Вълнообразни движения, които символизират телесните процеси и трансформацията. Подсъзнателно асоциира продуктите с вътрешни промени.

**Характеристики:**
- Елегантни вълнообразни движения
- Асоциация с телесни процеси
- Професионален и научен вид

### 4. Transformation Grid
**Психологически ефект:** Организация, прогрес и структура

Структурирана мрежа с леко движение, която създава усещане за точност, наука и измерими резултати.

**Характеристики:**
- Дискретна, структурирана анимация
- Асоциации с наука и прецизност
- Подходяща за аудитория, която цени данни

### 5. Neuron Canvas (Legacy)
Оригиналната анимация с неврони и връзки. Запазена за обратна съвместимост.

### 6. None
Без анимация - за минималистичен дизайн или бавни устройства.

## Как да смените анимацията

### От Админ Панела

1. Отворете `admin.html`
2. Отидете в раздел "Глобални"
3. Натиснете "Редактирай" на настройките на сайта
4. Изберете желаната анимация от падащото меню "Анимация на фона"
5. Натиснете "Запиши"
6. Запишете промените в панела

### Програмно

Редактирайте `settings.background_animation` в `page_content.json`:

```json
{
  "settings": {
    "site_name": "BIOCODE",
    "site_slogan": "the science of life",
    "logo_url": "https://...",
    "background_animation": "flowing-energy"
  }
}
```

Възможни стойности:
- `flowing-energy` (препоръчително)
- `wellness-pulse`
- `metabolic-waves`
- `transformation-grid`
- `neuron-canvas`
- `none`

## Технически детайли

### Имплементация

Всички нови анимации (освен `neuron-canvas`) са имплементирани с CSS, което означава:
- Нулево натоварване на JavaScript
- Отлична производителност
- Автоматично адаптиране към светли и тъмни теми
- Респектиране на потребителските настройки за намалено движение

### Файлове

- `index.css` - CSS дефиниции на анимациите
- `index.js` - JavaScript логика за инициализация
- `index.html` - HTML контейнер за анимациите
- `admin.html` - Админ панел контрола

### Поддръжка на теми

Всички анимации автоматично се адаптират към:
- Светла тема (`data-theme="light"`)
- Тъмна тема (`data-theme="dark"`)

CSS променливите `--accent-rgb` се използват за динамично оцветяване според активната тема.

## Психологически принципи

### Изграждане на доверие
- Гладки, предвидими движения
- Професионален визуален стил
- Асоциации със здраве и наука

### Стимулиране на действие
- Достатъчно динамика за привличане на вниманието
- Не прекалено интензивни
- Подсъзнателни асоциации с енергия и трансформация

### Оптимизация за целевата аудитория
- Продукти за отслабване изискват баланс между доверие и стимул
- Анимациите избягват агресивност
- Фокус върху здравословна трансформация

## Best Practices

1. **За нови сайтове:** Започнете с `flowing-energy` - балансирана и ефективна
2. **За повече доверие:** Използвайте `wellness-pulse`
3. **За научна аудитория:** `transformation-grid` е най-подходяща
4. **За емоционална връзка:** `metabolic-waves` създава усещане за промяна

## Performance

Всички анимации са оптимизирани за:
- Минимално използване на CPU
- Плавна работа на 60fps
- Респектиране на `prefers-reduced-motion`
- Адаптивност към размера на екрана
